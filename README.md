# PCSX2 Game Cover Downloader

<img width="491" height="569" alt="image" src="https://github.com/user-attachments/assets/c50695ff-018f-4473-82fc-0423eb3876aa" />



Simple utility for downloading PS2 cover art for PCSX2.

## Cover Source
This tool downloads cover images from the **ps2-covers** repository by **xlenore**.

## Default Directories
- **Cache:** `Documents/PCSX2/cache/gamelist.cache`  
- **Covers:** `Documents/PCSX2/covers/`

You can also select folders manually.



## How to Use
1. Choose your `gamelist.cache` file  
2. Choose your covers output folder  
3. Click **Download**  



## Requirements
- Windows 10 or newer  
- .NET 8 Runtime
